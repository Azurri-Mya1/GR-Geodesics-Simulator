# General Relativity Geodesic Simulator

# Overview
This project is an interactive MATLAB simulation of particle motion in Schwarzschild spacetime.  
It models geodesic trajectories under varying mass and angular momentum using numerical methods.

# Features
- Real-time control of mass (M) and angular momentum (L) using sliders
- Numerical solution of geodesic equations using ODE45
- 3D visualization of Schwarzschild spacetime geometry
- Interactive UI for exploring gravitational effects on motion
- Dynamic plotting of particle trajectories

# Technologies
- MATLAB
- Numerical Methods (ODE solving)
- Scientific Computing & Visualization

# How It Works
The simulation solves the geodesic equations numerically and visualizes the resulting trajectory in a 3D representation of curved spacetime.  
Users can adjust parameters such as mass and angular momentum to observe how motion changes under different gravitational conditions.

# How to Run
1. Open MATLAB
2. Load or paste the script into a new file
3. Run the script
4. Use sliders to adjust parameters and observe changes in real time

# Purpose
This project was created to explore general relativity concepts through computational modeling and interactive visualization.
